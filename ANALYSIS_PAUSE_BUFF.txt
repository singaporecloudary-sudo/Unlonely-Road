================================================================================

【文件位置1】F:\游戏制作\合成飞车射击\config\game-config.js

【Buff配置信息】(行号: 362-375)

buffs: {
  // 火力增强BUFF
  fireBoost: { 
    multiplier: 2.0,      // 伤害倍率(2倍)
    duration: 5000,       // 持续时间 5000ms
    spawnChance: 0.08     // 生成概率 8%
  },
  
  // 护盾BUFF
  shield: { 
    damageReduction: 0.5, // 减伤比例 50%
    duration: 6000,       // 持续时间 6000ms
    spawnChance: 0.06     // 生成概率 6%
  },
  
  // 加速BUFF
  speedBoost: { 
    multiplier: 1.5,      // 速度倍率 1.5倍
    duration: 4000,       // 持续时间 4000ms
    spawnChance: 0.1      // 生成概率 10% (最高)
  },
  
  // 回血BUFF (不能叠加，直接回复)
  heal: { 
    hpRestoreRatio: 0.3,  // 回复比例 30% (相对maxHp)
    spawnChance: 0.07     // 生成概率 7%
  }
},

buffSpawnInterval: 8000   // BUFF道具生成间隔 8000ms

【Buff类型】
1. fireboost (火力增强/火力强化)
   - 效果: 子弹伤害翻倍
   - 持续时间: 5秒
   - 特殊: 会生成isPower标记的强化弹(渲染为烈焰彗星光束)
   - 生成概率: 8%

2. shield (护盾)
   - 效果: 受到伤害时减伤50%
   - 持续时间: 6秒
   - 视觉效果: 玩家周围显示蓝色护盾圆
   - 生成概率: 6%

3. speed (加速)
   - 效果: 玩家移动速度提升50%
   - 持续时间: 4秒
   - 生成概率: 10% (最容易获得)

4. heal (回血)
   - 效果: 立即恢复maxHp的30%
   - 不参与叠加计数
   - 生成概率: 7%

4. Buff视觉反馈总结
================================================================================

【获得BUFF时】
1. 浮动文字提示 (在玩家头顶)
   坐标: player.x, player.y - 90
   颜色: #69F0AE (亮绿色)

- 标签: "火力", "护盾", "加速"
- 实时显示: 只显示当前激活的BUFF

【强化弹视觉效果】

- isPower弹: 烈焰彗星光束
  颜色: 橙金→红色渐变 (#FF6D00 → #FF3C00)
  特效: 更宽的拖尾(55px), 多层光晕, 飞溅粒子
